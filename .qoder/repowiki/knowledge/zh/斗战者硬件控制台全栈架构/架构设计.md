- **后端枢纽**：`server/api/Program.cs` 作为核心启动器，注册 `HardwareAbstractionLayer`、`SmuController` 等单例服务，并托管 `TelemetryBackgroundService` 以维持 WebSocket 广播。
- **通信契约**：前后端通过 `/api/control` REST 接口下发指令，并通过 `ws://127.0.0.1:3100/ws` 建立毫秒级遥测数据同步通道。
- **自托管集成**：后端直接托管编译后的 React 前端（`wwwroot/`），并通过 WinForms Shell (`Douzhanzhe.Shell`) 利用 WebView2 嵌入页面，形成统一的桌面应用入口。
- **驱动桥接**：后端通过 P/Invoke 调用 `inpoutx64.dll` 绕过 OS 限制访问 EC 寄存器，并封装 `ryzenadj.exe` 子进程处理 SMU 调优。
