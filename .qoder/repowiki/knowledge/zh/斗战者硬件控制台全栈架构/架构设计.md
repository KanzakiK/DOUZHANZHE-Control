- **三层协同**：`Douzhanzhe.Shell` (WinForms) 作为宿主进程，负责启动 `Douzhanzhe.API` (.NET 8) 并通过 WebView2 加载其托管的 React 静态资源。
- **通信契约**：前端通过 `http://127.0.0.1:3100` 访问 RESTful API 进行控制，并通过 `/ws` 端点建立 WebSocket 连接获取实时遥测数据流。
- **硬件抽象**：后端通过 `Douzhanzhe.HAL` 库统一封装底层驱动（WinRing0, inpoutx64）及外部工具（ryzenadj, nvidia-smi），向上提供标准化的 EC、SMU 及 WMI 接口。
- **配置共享**：系统状态、UI 布局及自定义参数通过 JSON 文件在 `server/config` 目录下持久化，实现前后端状态的跨会话同步。
