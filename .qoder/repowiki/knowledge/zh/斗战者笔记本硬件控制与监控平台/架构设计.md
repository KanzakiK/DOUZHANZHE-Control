- **分层架构**：采用 C# Minimal API (:3100) 作为单一后端，前端 React 应用通过 `wwwroot/` 静态托管。
- **硬件抽象层 (HAL)**：`DriverBridge` 封装 `inpoutx64` 实现 EC 寄存器直写（风扇/背光）；`WmiInterface` 调用 ACPI `MiInterface` 处理 GPU 模式与系统锁；`SmuController` 通过子进程调用 `ryzenadj.exe` 管理 AMD SMU 功耗墙。
- **GPU 控制引擎**：`NvapiGpuController` 采用双层设计，优先加载蛟龙控制台 `KaronOC.dll` 绕过笔记本 NVAPI 限制实现超频，回退至标准 NVAPI P/Invoke 进行状态读取。
- **数据流**：`TelemetryBackgroundService` 以 250ms 间隔轮询硬件状态并通过 WebSocket 推送至前端；控制指令经去抖后通过 REST API 下发至对应控制器。
