- **单体后端架构**：采用 ASP.NET Core Minimal API 作为核心枢纽，集成 `HardwareAbstractionLayer` (EC 直写)、`SmuController` (RyzenAdj 子进程) 和 `WmiInterface` (系统管理)，提供 RESTful 接口与 WebSocket 实时遥测。
- **前后端分离与自托管**：前端 React 应用编译后由后端 `wwwroot/` 静态托管，并通过 `ws://127.0.0.1:3100/ws` 建立全双工通信通道，实现毫秒级数据同步。
- **桌面壳层集成**：可选的 WinForms Shell (`Douzhanzhe.Shell`) 通过 WebView2 嵌入前端页面，并利用 Windows Task Scheduler 实现开机自启与系统托盘最小化，形成完整的桌面应用体验。
- **驱动桥接机制**：后端通过 P/Invoke 调用 `inpoutx64.dll` 绕过操作系统限制直接访问 EC 寄存器，同时封装 `ryzenadj.exe` 和 `nvidia-smi` 等外部工具以扩展硬件控制能力。
