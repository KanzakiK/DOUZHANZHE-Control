- **分层架构**：`api` 层（ASP.NET Core Minimal API）负责 HTTP/WebSocket 通信与业务编排，`hal` 层（硬件抽象层）封装底层驱动交互。
- **依赖方向**：API 项目引用 HAL 库，HAL 通过 P/Invoke 调用 `inpoutx64.dll` 实现 Ring0 级物理内存/IO 端口访问。
- **核心组件**：`HardwareAbstractionLayer` 统一暴露 EC 寄存器读写；`SmuController` 通过子进程调用 `ryzenadj.exe` 调整 AMD CPU 功耗/频率；`WmiInterface` 利用 ACPI WMI 方法控制风扇与 GPU 模式；`NvapiGpuController` 处理 NVIDIA 显卡超频。
- **入口与配置**：`Program.cs` 为单一入口，注册单例服务并托管静态前端资源；配置通过 JSON 文件持久化至共享目录。
